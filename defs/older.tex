\section{Basic Concepts}

\bdefn
Let $S$ be a set. We say $S$ is \emph{countable} if there exists a bijection between $S$ and a subset of the natural numbers $\mathbb{N}$.
\edefn

\bexpl
This definition means that we can "count" the elements of $S$ using natural numbers, even if $S$ is infinite. Examples include $\Z$ and $\mathbb{Q}$, but not $\R$.
\eexpl

\btheorem
The set of integers $\Z$ is countable.
\etheorem

\bexpl
We can prove this by showing that the mapping $f: \mathbb{N} \to \Z$ defined by:
$$f(n) = \begin{cases} 
\frac{n}{2} & \text{if } n \text{ is even} \\
-\frac{n+1}{2} & \text{if } n \text{ is odd}
\end{cases}$$
is a bijection. This gives us the sequence: $0, -1, 1, -2, 2, -3, 3, \ldots$
\eexpl
